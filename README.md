# SAE-105-Traitement-des-données
Dans le cadre de la SAE 105 – Traitement des données, ce projet a pour objectif de mettre en pratique les méthodes de sélection, de pré-traitement et d’analyse de données à
partir de fichiers réels. Le sujet choisi, le projet n°18 : le solaire, vise à étudier l’importance de la production d’électricité d’origine solaire et de son stokage en
France.Pour mener cette étude, nous nous appuyons sur les données mises à disposition par RTE (Réseau de Transport d’Électricité) via le site officiel Analyses et données de
RTE. Ces données, accessibles sous forme de fichiers CSV, concernent la production d’électricité en France selon les différentes filières, et plus particulièrement la
production solaire photovoltaïque.Dans un premier temps, nous avons sélectionné les fichiers de données les plus pertinents pour notre problématique, notamment ceux portant sur la production solaire.Les fichier csv seront mis en évidence .
